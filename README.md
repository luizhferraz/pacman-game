# Pac-Man Game

Um clone simples do clássico jogo Pac-Man, desenvolvido usando HTML, CSS e JavaScript puros. O objetivo é comer todos os pontos do labirinto enquanto escapa dos fantasmas.

## 🎮 Funcionalidades

- Mapa labirinto inspirado na versão clássica.
- Controle do personagem Pac-Man com as teclas do teclado.
- Sistema de pontuação.
- Fantasmas que perseguem o jogador.
- Reinício de partida ao perder todas as vidas ou vencer.

## 🔧 Tecnologias

- **HTML**
- **CSS**
- **JavaScript** (Vanilla JS)

## 📦 Como Rodar Localmente

1. **Clone este repositório:**
    ```bash
    git clone https://github.com/luizhferraz/pacman-game.git
    ```
2. **Navegue até o diretório do projeto:**
    ```bash
    cd pacman-game
    ```
3. **Abra o arquivo `index.html` em seu navegador**

Pronto! O jogo Pac-Man estará disponível para jogar localmente.

## 🚦 Como Jogar

- Use as teclas **W**, **A**, **S**, **D** ou as setas do teclado para mover o Pac-Man.
- Coma todos os pontos para vencer.
- Evite ser pego pelos fantasmas — se perder todas as vidas, o jogo reinicia.
